You can run this project locally using Python or Node.js.

### Option 1: Using Python (Recommended)
No installation required on macOS/Linux.

1.  Open your terminal in this folder.
2.  Run the command:
    ```bash
    python3 -m http.server 8000
    ```
    OR simply use the npm script if you have Node:
    ```bash
    npm run dev
    ```
3.  Open browser: [http://localhost:8000](http://localhost:8000)

### Option 2: Using Node.js

1.  Install dependencies (optional, just for `serve`):
    ```bash
    npm install
    ```
2.  Start the server:
    ```bash
    npm start
    ```
3.  Open browser: [http://localhost:3000](http://localhost:3000)

### Option 3: VS Code (Easiest)
1.  Open this folder in VS Code.
2.  Press `Cmd+Shift+P` -> Type "Run Task" -> Select **"Run Python Server"**.
3.  A terminal will open and the server will start.
